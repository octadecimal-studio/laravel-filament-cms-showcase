# Laravel Filament CMS Showcase

Sanitized public mirror of a private CMS project. It keeps the parts that are
useful for technical review while removing private client data, media, secrets,
environment files, database backups and deployment-specific instructions.

## What This Demonstrates

- Laravel 12 application code and testing structure
- Filament resources, panels, permissions and media-library integration
- Livewire-oriented CMS interactions
- Google Calendar integration work in the retained commit history
- Tailwind, PostCSS, Vite and Axios in a Laravel frontend pipeline
- Docker/NGINX development infrastructure
- Preserved Git history for the retained source paths

## Sanitization Notes

The public mirror is not intended to be a directly deployable client project.
It is a portfolio-safe code sample with historical commits preserved wherever
the corresponding source paths were retained.
